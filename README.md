# Quick Start

1. `npm install` 로 종속성 설치.

2. `npm run serve` 로 실행.

3. `localhost:3000` 에 접속

4. 번역할 영상을 추가하고, 자막을 입력한 뒤 공유할 수 있습니다.

5. 공유 링크는 `8192`길이까지 가능하며, 넘어가는 경우는 범위를 지정해서 공유하거나, 최대로 가능한 부분까지 공유가 가능합니다.
